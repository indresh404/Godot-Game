# 🎮 3-Player Mario-Type Platformer Game (Godot)

## 📅 Project Finished
June 2025

## 📌 Project Overview
This project is a **2D platformer game** developed using the **Godot Game Engine**.  
The game is inspired by classic **Mario-style gameplay**, where players move through levels, collect coins, avoid enemies, and reach the final goal.

The project focuses on implementing **game physics, player interaction, enemy mechanics, scoring systems, and victory/defeat conditions**.

---

## 🎮 Game Features

### 👥 Multiple Playable Characters
The game supports **3 different playable characters**.  
Each character represents a unique personality and adds variety to the gameplay experience.

### 🕹️ Platformer Mechanics
The game includes typical **side-scrolling platformer mechanics**:
- Player movement (left/right)
- Jumping using physics-based motion
- Gravity and collision detection
- Interaction with platforms and objects

### 👾 Enemies / Villains
The game includes **villain characters** that challenge the player during gameplay.
- Enemies move within defined areas
- Collision with enemies can cause the player to lose the game
- Enemies increase the difficulty of the level

### 🪙 Coin Collection System
A **coin collection system** is implemented:
- Coins are placed throughout the level
- Player collects coins while moving
- Coins increase the **score counter**
- Score is displayed during gameplay

### ⚙️ Physics System
The game uses **Godot's built-in physics system**, including:
- Gravity
- Collision detection
- Player movement physics
- Interaction with platforms and obstacles

### 🏆 Victory Condition
The player wins the game by:
- Successfully navigating the level
- Avoiding enemies
- Reaching the **final goal area**

When the goal is reached, a **Victory Scene** is displayed.

### ❌ Losing Condition
The player loses if:
- They collide with enemies
- Fall into a dangerous area

When this happens, a **Game Over / Losing Scene** appears.

---

## 🧩 Core Gameplay Flow
1. Player selects a character.
2. The level begins with movement and exploration.
3. Player collects coins to increase score.
4. Player avoids or navigates around enemies.
5. Player reaches the final goal.
6. Game displays **Victory Scene** or **Game Over Scene**.
